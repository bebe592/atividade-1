# Exercícios de Lógica com JavaScript 🚀

Este repositório contém exemplos práticos de fundamentos do JavaScript, focando em operações matemáticas, comparadores e manipulação de strings.

## 📝 O que este projeto cobre:

### 1. Operações Aritméticas
Cálculos básicos utilizando os operadores padrão:
- Soma (`+`)
- Subtração (`-`)
- Multiplicação (`*`)
- Divisão (`/`)

### 2. Operadores de Comparação
Testes lógicos para verificar relações entre valores:
- Igualdade (`==`) e Diferença (`!=`)
- Maior que (`>`) e Menor que (`<`)
- Maior ou igual (`>=`) e Menor ou igual (`<=`)

### 3. Operador Ternário
Exemplos de tomada de decisão concisa (if/else de uma linha) para verificar status como aprovação escolar ou maioridade.

### 4. Template Literals
Uso de crases (`` ` ``) para interpolação de variáveis e expressões dentro de strings, facilitando a leitura do console.

## 💻 Como executar
1. Certifique-se de ter o [Node.js](https://nodejs.org) instalado.
2. Clone o repositório ou copie o código para um arquivo chamado `index.js`.
3. No terminal, execute:
   ```bash
   node index.js
